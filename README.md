# dao-docs文档微博

## 介绍
用来演示和记录相关知识的微博文档

## 软件架构
采用vitepress框架编写的文档、微博框架

## 安装教程

1.  `npm install -D vitepress` 安装框架
2.  `npx vitepress init` 初始化项目
3.  `npm run docs:dev` 启动项目
4.  `npm run docs:build` 打包项目

## 使用说明

1.  .gitignore 忽略开发缓存不上传gitee `docs/.vitepress/cache/`
2.  start.bat 启动项目
3.  build.bat 打包项目
